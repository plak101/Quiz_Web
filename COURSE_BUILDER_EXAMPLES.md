## Workflow Examples

### Workflow 1: Draft ? Review ? Publish
```
1. Create course with IsPublished = false
2. Add all chapters, lessons, contents
3. Review in Step 4 (Preview)
4. Go back to Step 1, check "Xu?t b?n ngay"
5. Save again with IsPublished = true
```

### Workflow 2: Quick Publish
```
1. Fill all required info in Step 1
2. Check "Xu?t b?n ngay"
3. Add structure in Step 2
4. Add contents in Step 3
5. Publish immediately in Step 4
```

### Workflow 3: Incremental Building
```
1. Create basic course structure
2. Save as draft
3. Come back later, add more lessons
4. Save as draft again
5. Add lesson contents gradually
6. Finally publish when complete
```

### Workflow 4: Edit Published Course
```
1. Navigate to /courses/builder?id={courseId}
2. Modify any step
3. Save changes
4. Course remains published
```

## Tips & Best Practices

### Content Organization
1. Start with a free preview lesson (Public)
2. Group related lessons into logical chapters
3. End each chapter with a comprehensive test
4. Use Private visibility for advanced/bonus content

### Lesson Content Mix
- 40% Theory (Text + Images)
- 30% Video (Demonstrations)
- 20% FlashcardSet (Key concepts)
- 10% Test (Assessment)

### Pricing Strategy
- Free courses: Good for building audience
- 99,000 - 299,000 VN?: Short courses (10-20 lessons)
- 499,000 - 999,000 VN?: Medium courses (30-50 lessons)
- 1,000,000+ VN?: Comprehensive courses (50+ lessons)

### Publishing Strategy
1. Create complete structure first (draft)
2. Add at least 30% of content
3. Publish as "Early Access"
4. Add remaining content progressively
5. Update course regularly
